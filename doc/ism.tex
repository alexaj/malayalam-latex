Separate commands are provided for changing the fonts
inside any of the above options. For example,
if you are using `\texttt{karthika}' and want to
use `\texttt{bhavana}' for some text, simply
use the command \verb"\bhavana" before the text.
\verb"\revathi", \verb"\gopika", \verb"\kaumudi"  etc. also work.
